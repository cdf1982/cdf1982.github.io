Powerful Features for Every Workflow

Workflow Automation

Create custom workflows that automatically handle your scanned documents:

  • Use system share sheet for maximum flexibility
  • Rename with intelligent templates and tokens
  • Chain multiple actions in exact order
  • Optional auto-delete after workflow completes

OCR & Searchable PDFs

Make your scans searchable and extract text automatically:

  • Automatic OCR creates searchable PDF layers
  • Per-word accurate text positioning
  • Companion .txt file for every OCR'd scan
  • Built-in text viewer with search and highlighting
  • Manual OCR for existing PDFs
  • Attach text files in workflow actions
  • Adjustable quality for optimal file sizes
